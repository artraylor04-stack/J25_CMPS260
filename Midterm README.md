#Tech Store Discount Calculator

#Purpose of the app 
It is a program that calculates a customer’s invoice at a tech store based on purchase amount and membership status. It displays: subtotal,discounts applied, and final total. Also memberships and how members receive an additional 5% discount.

#Execution 
My code can be run using Visual Studio. Using Visual Studio or VS Code, simply build and run the project.
